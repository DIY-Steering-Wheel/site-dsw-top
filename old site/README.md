# Old Site
Arquivos da implementacao antiga em Next.js movidos para esta pasta.

## Conteudo
- `src/`: paginas e componentes da versao anterior
- `package.json`: dependencias e scripts antigos
- configs do Next, TypeScript, Tailwind e ESLint

## Como rodar a versao antiga
1. Entre nesta pasta.
2. Instale as dependencias se necessario.
3. Rode `npm run dev`.
